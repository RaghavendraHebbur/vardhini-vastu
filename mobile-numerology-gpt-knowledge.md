## STEP 1 — HOW TO CALCULATE MOBILE TOTAL

**Rule:** Add all digits of the 10-digit mobile number together. If result is double digit, reduce again.

**Formula:**
Mobile Total = Sum of all digits -> reduce to single digit (1-9)

**Example:**
Mobile: 9 5 9 9 5 5 6 6 4 2
Sum = 9+5+9+9+5+5+6+6+4+2 = 60
6+0 = 6
Mobile Total = 6

**Important Note:**
- Always use the **original mobile number as assigned** (10 digits, India format)
- Do NOT add country code (+91) to the number
- If number starts with 0, include the 0 in calculation

---

## STEP 2 — HOW TO CALCULATE BIRTH NUMBER (BN / MULANK)

**Rule:** Add only the **day of birth** digits. Reduce to single digit.

**Formula:**
BN = Day digits reduced to single digit

**Examples:**
DOB: 05-08-1988 -> BN = 0+5 = 5
DOB: 11-03-1990 -> BN = 1+1 = 2
DOB: 29-07-1975 -> BN = 2+9 = 11 -> 1+1 = 2
DOB: 06-02-1986 -> BN = 0+6 = 6

---

## STEP 3 — HOW TO CALCULATE DESTINY NUMBER (DN / BHAGYANK)

**Rule:** Add ALL digits of full date of birth (DD+MM+YYYY). Reduce to single digit.

**Formula:**
DN = Sum of (day + month + year digits) -> reduce to single digit

**Examples:**
DOB: 05-08-1988
= 0+5+0+8+1+9+8+8 = 39 -> 3+9=12 -> 1+2 = 3
DN = 3

DOB: 06-02-1986
= 0+6+0+2+1+9+8+6 = 32 -> 3+2 = 5
DN = 5

### Calculate Kua Number

The Kua (Gua) number is derived from the birth year and gender. It determines 8 auspicious and inauspicious directions for a person.

**For Men:**
1. Take last two digits of birth year
2. Add the digits together
3. Keep adding until single digit (1–9)
4. Subtract from 10 = Kua number

**Example (Male, 1985):** 8 + 5 = 13 → 1 + 3 = 4 → 10 - 4 = **Kua 6**

**For Women:**
1. Take last two digits of birth year
2. Add the digits together
3. Keep adding until single digit
4. Add 5 = Kua number (if result > 9, add digits again)

**Example (Female, 1985):** 8 + 5 = 13 → 1 + 3 = 4 → 4 + 5 = **Kua 9**
